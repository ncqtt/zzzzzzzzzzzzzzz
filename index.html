KernelForge

Build Android kernels in the cloud. Connect your GitHub account to fork repositories, configure toolchains, and dispatch builds — all from one interface.

01 Authenticate with GitHub OAuth
02 Fork any kernel repository to your account
03 Configure toolchain, defconfig, and outputs
04 Dispatch GitHub Actions build and monitor logs

KernelForge uses GitHub OAuth to read your repositories and dispatch Actions workflows on your behalf. No code is stored on our servers — builds run in your own GitHub Actions environment.

Dashboard / Repositories

Fork a kernel repository

Search public kernel repositories and fork them to your GitHub account. The workflow file will be added automatically.

Workflow injection
When you fork a repository, KernelForge automatically adds .github/workflows/build.yml to the fork. Your secrets (TELEGRAM_TOKEN, TELEGRAM_CHAT_ID) must be added separately in the repository settings.
